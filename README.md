## 2. Cách chạy dự án

1. Mở terminal tại thư mục này, cài đặt thư viện:
   npm install

2. Khởi động Expo:
   npx expo start

3. Quét mã QR bằng ứng dụng Expo Go trên điện thoại, hoặc nhấn 'a' để mở trên Android Emulator, 'i' để mở trên iOS Simulator.
